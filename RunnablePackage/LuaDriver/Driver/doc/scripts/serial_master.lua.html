Script serial_master.lua

serial_master.lua is an optional script to manage the DUT UART connection and publish the log over ZMQ.

It continuously publishes the UART traffic to a ZMQ socket (DUT_UART_ZMQ), and takes requests to send text over UART to the DUT using another ZMQ socket (UART_ZMQ_REQ).

It uses the following ZMQ Sockets:

  • DUT_UART_ZMQ PUB socket
  • UART_ZMQ_REQ REP socket

Usage:

  • serial_master.lua [-c config file] [-u DUT #] [OPT=VAL]
  • serial_master.lua -c config.zmq -u 0 UNIT_STAGE=PRESMT2
